# 6. Conclusion

We have demonstrated that real-time phishing detection using URL feature extraction and Random Forest classification can substantially outperform traditional blacklist-based approaches, achieving 97.4% F1-score on a balanced dataset of 30,000 URLs. Unlike blacklist methods, which fail to detect phishing sites created within the past 4–8 hours, PhishGuard operates without external database queries and produces classification decisions in under 50 milliseconds — making it suitable for browser-level deployment. Our ablation study reveals that URL lexical features contribute most significantly to detection performance, while the Random Forest's feature weighting mechanism provides a meaningful 3.1% improvement over an unweighted baseline.

Our study has several limitations. First, the dataset consists exclusively of English-language URLs; performance on phishing campaigns targeting Vietnamese-language banking and e-commerce platforms remains unvalidated. Second, PhishGuard relies on static URL and page features captured at a single point in time; techniques that adapt to evolving phishing patterns over time — such as online learning or concept drift detection — are not explored. Third, the HTML and JS feature extraction steps introduce latency beyond the URL-only pipeline, and the trade-off between feature richness and inference speed was not fully characterized.

Future work will address these limitations along three directions. First, we plan to construct a Vietnamese-language phishing dataset in collaboration with local cybersecurity agencies to evaluate and adapt PhishGuard for the Vietnamese internet ecosystem. Second, we will explore online learning variants that update the classification model incrementally as new phishing samples are confirmed, enabling adaptation to emerging attack patterns. Third, we will investigate lightweight feature subsets that maintain accuracy above 95% while reducing inference latency to under 5 milliseconds, making PhishGuard viable for deployment on mobile devices and low-power network appliances.
